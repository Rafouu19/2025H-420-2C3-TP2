## Description
Cette application gère des tournois de jeux vidéo (1 vs 1) :
- Création de tournois
- Chargement de joueurs et de matchs depuis des CSV
- Saisie des résultats
- Affichage du classement
- Sauvegarde en JSON
- Génération d'un rapport texte

## Prérequis
- Python 3.7+
